## Express app for users auth and posts

Auth service for users and their posts

#### Clone & install

* Clone this repo `https://github.com/swooosh13/express-users-auth.git`
* `cd express-users-auth`
* Run `npm install`
* Run `npm start`

### Stack

* PostgreSQL
* Sequelize
* Express
* Passport JS
* jsonwebtoken
* bcrypt
